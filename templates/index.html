Sobre mim

Iniciei como TI, evoluindo para a analista de logística e, em seguida, embarquei no setor Offshore. Mesmo sabendo que meu sonho de trabalho estava no desenvolvimento de softwares, precisei seguir para a área técnica, assim aliando um salário melhor a ter tempo com minha família.

Agora, como engenheiro de software em formação, minha paixão pela programação se une à criação 3D. Tenho habilidades em Python para automatizar processos empresariais, ambientes WEB e experiência em criação de ambientes 3D com Sketchup.

Meu objetivo é criar soluções inovadoras que impactem positivamente a vida das pessoas, combinando minha expertise em programação e modelagem 3D.

Serviços

Alguns tipos de trabalhos que venho desenvolvendo.

AUTOMATIZAÇÃO DE PROCESSOS (HIPERAUTOMAÇÃO) RPA

Analiso processos diários ou rotineiros. A partir daí crio soluções em softwares com interface amigável para que seja feito com pouca ou nenhuma ação humana.

DESENVOLVIMENTO DE APLICAÇÕES DESKTOP

Criação de softwares personalizados e feitos com cuidado para atender as necessidades. Sempre focado em aparar as arestas e manter o programa sem processos demorados.

MODELAGEM 3D E PLANTA HUMANIZADA

Criação de modelos 3D a partir de plantas, idéias e/ou desenhos estruturais. Com a opção de criação de render humanizado, ou seja, simulando a realidade.

POWER POINT MODERNO

Trasformo aquela apresentação sem graça em algo digno de uma apresentação moderna, sem a utilização de animações, caso você tenha pensado nisso.

Ciência de dados

Crio relatórios com base em gestão de Banco de dados integrado, auxiliando em tomadas de decisão a partir do tratamento de dados.

RASPAGEM DE DADOS

Crio serviços de envio de relatórios automáticos personalizados por e-mail com informações diversas disponíveis na internet, para acompanhamento.
